# UX/UI QA Test Series

Date: 2026-05-26

## Scope

Resume and portfolio pages for the public `cv-2026` site.

## Tests

1. Resume reading order
   - Expected flow: summary, core competencies, professional affiliation, career history, projects, education, certifications, additional information, online pages.
   - Result: passed.

2. AITAI placement
   - Expected: AITAI Associate appears as one separate professional affiliation line between core competencies and career history.
   - Result: passed.

3. Contact and interaction cues
   - Expected: email, LinkedIn, availability, resume link, and portfolio link are visible from the header.
   - Result: passed.

4. Resume and portfolio separation
   - Expected: resume carries credentials and formal profile information. Portfolio carries selected projects and a clear path back to the resume.
   - Result: passed.

5. SEO and AI discovery basics
   - Expected: robots sitemap reference, XML sitemap, canonical pages, JSON-LD, and AI-readable `llms.txt`.
   - Result: passed.

6. Browser smoke test
   - Expected: local Chrome renders the updated resume without the old combined credentials card, and the public contact paths remain easy to find.
   - Result: passed.
